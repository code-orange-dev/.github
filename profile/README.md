# Code Orange Dev School

**Asia's Bitcoin Developer Pipeline** — training developers across Southeast Asia to contribute to Bitcoin open-source.

### The Problem

[~41 developers](https://www.bitget.com/news/detail/12560605101959) power the $1.7 trillion Bitcoin Core codebase. Across Southeast Asia — 700 million people living under currency debasement, capital controls, and financial surveillance — that number is close to zero.

### What We Do

Code Orange is a Bitcoin-only, open-source developer education program based at [Bitcoin House Bali](https://coinfinity.co/en/blog/bitcoin-house-bali-indonesia). We take people from "curious Bitcoiner" to "open-source contributor" through structured cohorts, hands-on workshops, and a fellowship program that funds developers and educators working on Bitcoin.

**The pipeline:**

```
Reading Club → Sovereign Bitcoiner → Bitcoin Dojo → rawBit → Decoding Bitcoin → Fellowship → OSS Contribution
```

All curriculum is **free, open-source, and CC0-licensed**. Every graduate is a permanent asset for Bitcoin.
